# 2-Minute Demo Script — SentinelScan

**Time target:** 2 minutes

1. **Hook (0:00 - 0:15)**
   “Developers ship APIs fast and security gets left behind. SentinelScan gives teams instant OWASP risk visibility without heavy tooling.”

2. **Show the UI (0:15 - 0:40)**
   “This dashboard is built for speed: enter any REST endpoint and we run async checks for BOLA, auth, data exposure, rate limiting, SQLi/SSRF, and security headers.”

3. **Run Demo Mode (0:40 - 1:10)**
   “Toggle Demo Mode, click Run Scan. In demo mode, we scan an intentionally vulnerable endpoint so the findings light up immediately.”

4. **Highlight Findings (1:10 - 1:35)**
   “Each finding includes severity, CVSS-style score, evidence, and remediation guidance. This is what an engineer needs to fix issues fast.”

5. **Export Reports (1:35 - 1:50)**
   “We can export JSON for automation or PDF for audits and compliance.”

6. **Close (1:50 - 2:00)**
   “SentinelScan is lightweight, production-ready, and built for real-world impact. It fits into CI, pentest workflows, and dev teams shipping weekly.”
